# Contemplar CRM

Sistema de Gestão Comercial de Consórcios — desenvolvido para potencializar vendas e relacionamento com clientes.

## Sobre o Projeto

**Contemplar CRM** é uma plataforma SaaS para gestão de consórcios, com módulos de:
- Dashboard de performance e metas
- Funil de vendas
- Gerenciamento de Leads
- Carteira de Clientes
- Módulo de Inadimplentes
- Simulador de Consórcio
- Fila Inteligente
- Relatórios e Indicadores

**Domínio**: https://contemplarcrm.com.br *(em breve)*

## Tecnologias

- React + TypeScript
- Vite
- Supabase (banco de dados e autenticação)
- Tailwind CSS + shadcn-ui
- Recharts

## Desenvolvimento Local

```sh
# 1. Clone o repositório
git clone <URL_DO_REPOSITORIO>

# 2. Entre na pasta do projeto
cd simulaja

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento
npm run dev
```

## Deploy

O projeto é implantado automaticamente via **Vercel** a cada push na branch `main`.
